# 安装
## windows
文件下载安装
[下载](https://yarnpkg.com/latest.msi)

Chocolatey windowns包管理器下载
>   choco install yarn

Scoop 是一个 Windows 的命令行安装程序
>   scoop install yarn

如果 Node.js 没有安装，scoop 将给你一个建议来安装它。 例如：
>   scoop install nodejs

## node安装
>   node install -g yarn
